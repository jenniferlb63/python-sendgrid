**This helper is a stand alone module to help consume and process Inbound Parse data.**

# Quick Start

```python
python sendgrid/helpers/inbound/app.py
```

In another terminal:

```python
python sendgrid/helpers/inbound/send.py
```
View the results in the first terminal
